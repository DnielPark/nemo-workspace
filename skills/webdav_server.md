# WsgiDAV WebDAV 서버 (8080)

## 개요

내부 네트워크 WebDAV 공유 시스템. 사내 문서 공유용.

- **서비스:** WsgiDAV (WebDAV)
- **포트:** 8080 (내부망 전용, 외부 노출 금지)
- **프로세스:** `pythonw.exe` (PID: 5564)

> ⛔ **절대 건드리지 말 것!**
> - 수정 / 재시작 / 중단 금지
> - 사내 공유 시스템으로 중요 서비스

## Windows에서 WebDAV 마운트

```cmd
net use Z: http://192.168.1.211:8080
```

## 접속 확인

브라우저에서 `http://192.168.1.211:8080/` 접속

## 상태 확인

```cmd
netstat -ano | findstr :8080
```

## 참고

- 이 서비스는 내부망 전용이며 외부에서 접근 불가
